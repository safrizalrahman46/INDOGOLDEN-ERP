# INDOGOLDEN-ERP


INDOGOLDEN-ERP adalah aplikasi Enterprise Resource Planning berbasis web yang dibangun menggunakan **Laravel**, **Filament Admin Panel**, dan **PostgreSQL**. Aplikasi ini dirancang untuk membantu proses operasional bisnis secara terpusat, rapi, dan mudah dikembangkan.

## Teknologi yang Digunakan

- **Laravel**  
  Framework backend utama untuk mengelola routing, business logic, authentication, API, dan struktur aplikasi.

- **Filament**  
  Admin panel modern berbasis Laravel untuk membuat halaman CRUD, dashboard, form, table, filter, dan manajemen data secara cepat dan interaktif.

- **PostgreSQL**  
  Database relasional yang digunakan untuk menyimpan data aplikasi secara aman, stabil, dan scalable.

## Fitur Utama

- Dashboard admin berbasis Filament
- Manajemen master data
- CRUD data operasional
- Role dan permission user
- Import dan export data
- Filter, search, dan pagination
- Validasi data
- Relasi antar modul
- Tampilan responsive untuk desktop dan tablet
- Struktur modular agar mudah dikembangkan

## Tujuan Aplikasi

Aplikasi ini dibuat untuk mendukung kebutuhan internal perusahaan dalam mengelola proses bisnis seperti:

- Inventory
- Gudang
- Cabang
- Request barang
- Pengiriman barang
- Master data
- Laporan operasional
- Perhitungan HPP
- Kontrol transaksi dan aktivitas user

## Requirement Sistem

Pastikan environment sudah memiliki:

- PHP >= 8.2
- Composer
- Node.js & NPM
- PostgreSQL
- Laravel
- Filament
- Git
